# Counter with redux-toolkit

This project demonstrates RTK fundamentals: <b>store</b>, <b>slice</b>,
<b>reducers</b>, and React bindings via <b>useSelector</b> 
<b>useDispatch</b>. 

![counter](https://github.com/user-attachments/assets/9234bb50-851f-43d9-bb00-961b4c595cba)

## Features

- Increment
- Decrement
- Reset
- IncrementByAmount

## How to run
clone project
```
npm install
npm run dev
```
